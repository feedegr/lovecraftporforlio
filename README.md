# lovecraftporforlio

A modern, responsive portfolio website showcasing projects, skills, and contact information.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Scrolling**: Navigation links smoothly scroll to sections
- **Modern UI**: Clean and professional design with gradient backgrounds
- **Interactive Elements**: Hover effects and scroll animations
- **Portfolio Sections**:
  - Hero section with call-to-action
  - About section
  - Projects showcase with three sample projects
  - Skills categorized by Frontend, Backend, and Tools
  - Contact information section

## Getting Started

### View the Portfolio

Simply open the `index.html` file in your web browser:

```bash
# Option 1: Open directly in browser
open index.html

# Option 2: Serve with a local server (recommended)
python3 -m http.server 8000
# Then visit http://localhost:8000 in your browser
```

## Customization

Edit the following files to personalize your portfolio:

- **index.html**: Update content, project descriptions, skills, and contact information
- **styles.css**: Modify colors, fonts, and styling
- **script.js**: Add or modify interactive features
